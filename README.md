Built using **AI-assisted vibe coding** (Claude · GitHub Copilot).

## Tech

- Pure HTML / CSS / Vanilla JS — no frameworks, no build step
- GitHub REST API for live repo data
- Hosted on Netlify / GitHub Pages

## Deploy

Just open `index.html` in a browser, or deploy the file to any static host:

```bash
# GitHub Pages
git push origin main

# Netlify drag & drop
# Drop index.html at app.netlify.com
```
